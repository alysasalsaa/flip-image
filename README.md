# flip-image

A 2x2 image matrix where each image can be flipped independently using arrow buttons.

## Features

- [x] Single image displayed in a 2x2 matrix
- [x] Flip any image vertically or horizontally using arrow buttons
- [x] **Bonus:** Enter a custom image URL to replace the default image
- [x] **Bonus:** Display button to load the new image
- [x] **Bonus:** Error message shown if the image URL is invalid or not found

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## Constraints

- No image packages or libraries used

## How to run

Open `index.html` in your browser — no build tools needed.
